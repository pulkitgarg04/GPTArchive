# GPTArchive

GPTArchive is a browser extension designed to facilitate the archiving and organization of ChatGPT conversations. It provides users with a robust set of tools to bookmark, tag, annotate, and manage their AI interactions, ensuring critical information is easily retrievable.

## Key Features
*   **Conversation Archiving**: Instantly save ChatGPT conversations with a single click.
*   **Advanced Organization**: Categorize saved chats using custom tags and personal notes for efficient retrieval.
*   **Comprehensive Dashboard**: A dedicated management interface to view, edit, and delete archived content.
*   **Search and Filtering**: Rapidly locate specific conversations through keyword search and tag-based filtering.
*   **Data Portability**: Import and export archived data in CSV format for backup or analysis.
*   **Cloud Synchronization**: Automatically sync saved bookmarks across all devices logged into the same browser account.
*   **Dark Mode Support**: A user interface optimized for visual comfort in low-light environments.

## Installation Guide

To install GPTArchive as a developer extension:

1.  **Download the Source Code**: Clone this repository or download the ZIP file from [here](https://gptarchive.vercel.app/install) and extract it to a local directory.
2.  **Access Extension Management**: Open your browser (Chrome, Edge, or Brave) and navigate to the extensions management page (e.g., `chrome://extensions/`).
3.  **Enable Developer Mode**: Toggle the "Developer mode" switch located in the top-right corner of the page.
4.  **Load the Extension**: Click the "Load unpacked" button and select the `extension` directory from the downloaded files.
5.  **Verification**: The GPTArchive icon should now appear in your browser toolbar.

## Usage Instructions

1.  Navigate to [ChatGPT](https://chatgpt.com/).
2.  Open a conversation you wish to archive.
3.  Click the GPTArchive extension icon in the browser toolbar.
4.  Enter a title, optional tags, and notes, then save the bookmark.
5.  To manage saved items, open the extension popup and click the settings icon to access the Dashboard.
